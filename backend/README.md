# Attendance Management Backend
A Spring Boot backend application using Maven, Spring Security (JWT), Spring Data JPA, and Hibernate.

## Setup Instructions

Make sure you have Java 21+ and Maven installed.

### Build the project:
`mvn clean install`

### Run the server
`mvn spring-boot:run`

Server typically starts on `http://localhost:8082`.

## Overview
Connects to the React frontend running on port 5173 to serve API endpoints.
