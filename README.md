# Attendance Management System

This is a comprehensive full-stack Attendance Management System featuring Student and Faculty portals. It handles authentication, attendance tracking, and reporting.

## Project Structure
- **/frontend**: React.js based user interface.
- **/backend**: Spring Boot Java backend.

## Features
- JWT based User Authentication
- Student Dashboard for tracking attendance logs and overall hours
- Faculty Dashboard for managing student check-in and check-out times
- Graceful error handling and intuitive UI design
